Ejercicio Práctico: Estilos CSS

Creación y aplicación de estilos CSS

Este ejercicio tiene como objetivo aprender a vincular una hoja de estilos externa al documento HTML y aplicar propiedades básicas de diseño.

Objetivo

Aprender a vincular una hoja de estilos externa al documento HTML y aplicar propiedades básicas de diseño (colores, tipografía y manejo de imágenes) para mejorar la presentación de una página web.

Tareas Realizadas

  • Crear hoja de estilos externa en la carpeta CSS
  • Enlazar la hoja de estilos al documento HTML
  • Aplicar color de fondo lightblue al body
  • Cambiar color de fuente a rojo en los títulos
  • Dar formato a la imagen (400px x 300px)
  • Centrar la imagen en la página
